# Middleware de Node

Le middleware Node de Qwik City vous permet de connecter Qwik City à un serveur Nodejs qui utilise les fonctionnalités communes du middleware. Certains serveurs Node incluent :

- [Express](https://expressjs.com/)
- [Connect](https://www.npmjs.com/package/connect)
- [Polka](https://www.npmjs.com/package/polka)

Vous trouverez ci-dessous un exemple d'utilisation du middleware Node intégré dans l'appli [Express](https://expressjs.com/) d'un utilisateur. Notez que l'utilisateur peut décider de l'ordre des middlewares, y compris la gestion des ressources statiques et la façon de traiter les réponses 404.

```typescript
import { createQwikCity } from '@builder.io/qwik-city/middleware/node';
import qwikCityPlan from '@qwik-city-plan';
import render from './entry.ssr';
import express from 'express';
import { fileURLToPath } from 'node:url';
import { join } from 'node:path';

// Répertoires où se trouvent les ressources statiques
const distDir = join(fileURLToPath(import.meta.url), '..', '..', 'dist');
const buildDir = join(distDir, 'build');

// Crée le middleware Qwik City
const { router, notFound } = createQwikCity({ render, qwikCityPlan });

// Crée le serveur express
const app = express();

// Utilise le routeur de pages et le endpoint de Qwik City
app.use(router);

// Gestionnaires de ressources statiques
app.use(`/build`, express.static(buildDir, { immutable: true, maxAge: '1y', index: false }));
app.use(express.static(distDir, { index: false }));

// Utilise le gestionnaire 404 de Qwik City
app.use(notFound);

// Démarre le serveur express
app.listen(3000);
```
