# Intégrations

Qwik est livré avec des intégrations intégrées pour faciliter la connexion à vos services préférés, l'utilisation de vos outils de style ou de test préférés et le déploiement de votre application vers des services communs tels que Cloudflare, Netlify ou Express server, et le [Générateur de site statique (SSG)](https://qwik.builder.io/qwikcity/static-site-generation/static-site-config/).

Une fois que vous avez créé une application Qwik City, exécutez la commande suivante pour ajouter de nouvelles intégrations :

```shell
npm run qwik add
```

Vous serez invité à sélectionner l'intégration que vous souhaitez ajouter. Une fois que vous aurez sélectionné une intégration, elle sera ajoutée à votre application et vous pourrez commencer à l'utiliser.
