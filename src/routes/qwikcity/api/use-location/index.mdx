# `useLocation()`

Utilisez la fonction `useLocation()` pour récupérer un objet `RouteLocation` pour l'emplacement actuel.

```tsx
export interface RouteLocation {
  readonly params: RouteParams; // Record<string,string>
  readonly href: string;
  readonly pathname: string;
  readonly query: Record<string, string>;
}
```

La valeur de retour de `useLocation()` est similaire à celle de `document.location`, mais elle est sécurisée pour être utilisée sur le serveur où il n'y a pas d'objet global `location`.

## Paramètres de route

`useLocation()` encode les [Paramètres de route](../../../qwikcity/routing/route-parameters/index.mdx) comme paramètres.

Supposez que vous ayez fait :

- Route : `https://example.com/sku/[skuId]`
- L'utilisateur navigue vers : `https://example.com/sku/1234`
- Ensuite, le `skuId` peut être récupéré via `useLocation().params.skuId`.

```typescript
export default component$(() => {
  const { pathname, params } = useLocation();

  return (
    <>
      <h1>SKU</h1>
      <p>pathname: {pathname}</p>
      <p>skuId: {params.skuId}</p>
    </>
  );
});
```

Le code ci-dessus génère :

```html
<h1>SKU</h1>
<p>pathname: /sku/1234</p>
<p>skuId: 1234</p>
```
