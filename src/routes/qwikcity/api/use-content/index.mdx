# `useContent()`

La fonction `useContent()` récupère les informations de contenu les plus proches pour la route actuelle. L'objet retourné comprend :

```ts
headings: ContentHeading[] | undefined;
menu: ContentMenu | undefined;
```

Le tableau `headings` contient des données sur les éléments de titre html `<h1>` à `<h6>` d'un fichier markdown (https://developer.mozilla.org/fr/docs/Web/HTML/Element/Heading_Elements).

Les menus sont des données contextuelles déclarées avec des fichiers `menu.md`. Voir [définition du fichier menu](../../../qwikcity/content/menu/index.mdx) pour plus d'informations sur le format et l'emplacement du fichier.
