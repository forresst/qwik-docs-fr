# Démarrage avec Qwik

Qwik est un nouveau type de framework qui est [résumable](../concepts/resumable/index.mdx) (pas de JS et pas d'hydratation), construit pour le Edge et [familier pour les développeurs React](../cheat/qwik-react/index.mdx).

- [Stackblitz Qwik + Vite](https://qwik.new)
- [Exemples de terrains de jeux](/examples/reactivity/counter/)

## Prérequis

- [node.js v16](https://nodejs.org/en/download/) ou supérieur
- Votre IDE préféré ([vscode](https://code.visualstudio.com/) est recommandé)
- Commencez par lire [Pensez Qwik](../think-qwik/index.mdx)


## Création d'une appli à l'aide du CLI

La première étape est de créer une application. Qwik est livré avec un CLI qui vous permet de créer un squelette de base d'une application. Nous allons utiliser le CLI pour créer un _starter_ vierge afin que vous puissiez vous familiariser rapidement avec lui.

### Exécuter le CLI de Qwik dans votre shell

```shell
npm create qwik@latest
```

Le CLI vous guidera à travers un menu interactif pour définir le nom du projet et sélectionner l'un des _starters_ :

À ce stade, vous aurez le répertoire `qwik-app`, qui contient l'appli de démarrage.

## Exécution en mode développement

Une fois que l'application est chargée.

1. Allez dans le répertoire créé par `npm create qwik@latest`.

```shell
cd qwik-app
```

2. Installez les modules NPM :

```shell
npm install
```

3. Appelez le serveur de développement

```shell
npm start
```

4. Vous devriez voir un serveur fonctionner avec votre application To-do.

```shell

  VITE v3.1.1  ready in 140 ms

  ➜  Local:   http://localhost:5174/
  ➜  Network: use --host to expose

```

5. Visitez http://localhost:5174/ pour découvrir l'appli.

## Commandes

Voici quelques commandes utiles pour vous aider à démarrer. Pour une liste complète, veuillez vous référer au `package.json`.

- `npm start` : alias pour `npm run dev`(par défaut, il s'agit d'exécuter du SSR)
- `npm run preview` : Construction de la production et démarrage du serveur de prévisualisation.
- `npm run build` : construit l'application pour la production.
- `npm run qwik add` : Sélectionne une intégration à ajouter.
