# Démarrage avec Qwik

Qwik est un nouveau type de framework qui est [résumable](../concepts/resumable/index.mdx) (pas de JS et pas d'hydratation), construit pour le Edge et [familier pour les développeurs React](../cheat/qwik-react/index.mdx).

- [Stackblitz Qwik + Vite](https://qwik.new)
- [Exemples de terrains de jeux](/examples/reactivity/counter/)

## Prérequis

- [node.js v16](https://nodejs.org/en/download/) ou supérieur
- Votre IDE préféré ([vscode](https://code.visualstudio.com/) est recommandé)
- Commencez par lire [Pensez Qwik](../think-qwik/index.mdx)

## Création d'une appli à l'aide du CLI

La première étape est de créer une application. Qwik est livré avec un CLI qui vous permet de créer un squelette de base d'une application. Nous allons utiliser le CLI pour créer un _starter_ vierge afin que vous puissiez vous familiariser rapidement avec lui.

### Exécuter le CLI de Qwik dans votre shell

```shell
npm create qwik@latest
```

Le CLI vous guidera à travers un menu interactif pour définir le nom du projet et sélectionner l'un des _starters_ :

Après la création de votre nouvelle appli, vous verrez un affichage comme le suivant dans votre terminal :

```shell
💫 Let's create a Qwik app 💫

✔ Project name … qwik-app
✔ Select a starter › Basic

🦄  Success!  Project created in portfolio directory

🐰 Next steps:
   cd qwik-app
   npm install
   npm start

💬 Integration? Add Netlify, Cloudflare, Tailwind...:
   npm run qwik add

💬 Questions? Start the conversation at:
   https://qwik.builder.io/chat
   https://twitter.com/QwikDev

📺 Presentations, Podcasts and Videos:
   https://qwik.builder.io/media/
```

À ce stade, vous aurez le répertoire `qwik-app`, qui contient l'appli de démarrage.

## Exécution en mode développement

Une fois que l'application est chargée.

1. Allez dans le répertoire créé par `npm create qwik`.

```shell
cd qwik-app
```

2. Appelez le serveur de développement

```shell
npm start
```

4. Vous devriez voir un serveur fonctionner avec votre application To-do.

```shell

  VITE v3.1.1  ready in 140 ms

  ➜  Local:   http://localhost:5174/
  ➜  Network: use --host to expose

```

5. Visitez http://localhost:5174/ pour découvrir l'appli.

## Exécution en production

Pour exécuter l'application localement en production, exécutez la commande suivante :

```shell
npm run preview
```

Il construira votre application et le [serveur de prévisualisation Vite](https://vitejs.dev/config/preview-options.html) pour servir l'application.

## Déploiement

Il est très facile de déployer votre application Qwik vers le cloud, en tant que fichiers statiques, ou votre propre serveur.

Veuillez consulter la section [déploiement](/qwikcity/adaptors/overview/index.mdx) pour plus de détails.

## Autres intégrations

Qwik est conçu pour être intégré avec d'autres outils. Comme `vitest`, `tailwind`, `partytown`... lancez la commande suivante pour voir la liste des intégrations disponibles :

```shell
npm run qwik add
```

et n'oubliez pas de consulter la section [intégrations](/qwikcity/integrations/overview/index.mdx) pour plus de détails.
