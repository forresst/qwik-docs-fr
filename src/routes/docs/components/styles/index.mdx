## `useStyles$()`

Une référence chargée de manière différée pour les styles du composant.

Les styles de composants permettent à Qwik de charger de manière différée les informations de style pour le composant seulement quand c'est nécessaire. (Et d'éviter le double chargement en cas d'hydratation du SSR).

```tsx
import styles from './code-block.css?inline';
export const CmpStyles = component$(() => {
  useStyles$(styles);
  return <>Un peu de texte</>;
});
```
