# Ressources statiques

Utilisez le dossier `/public` pour stocker toutes les ressources statiques telles que les images, les polices, etc.

Tous les ressources statiques sont servis à partir du dossier racine de votre serveur (`/` tel que `https://example.com/`) et sont superposés à vos routes (les routes sont prioritaires).

```bash
public/
├── favicon.ico           # https://example.com/favicon.ico
└── images/
    └── logo.png          # https://example.com/images/logo.png
```
