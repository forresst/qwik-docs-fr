Les applications Web sont construites à partir de composants de la même manière que les applications générales sont construites à partir de fonctions.

La composition de fonctions ne serait pas très utile si vous ne pouviez pas passer des paramètres. De la même manière que les fonctions ont des paramètres, les composants ont des « props ». Un composant utilise les props pour transmettre des données à ses composants enfants.

> **Votre tâche** : Modifiez le code pour que l'appli parent `<App>` puisse passer `salutation="Hello"` et `name="World"` au `<Greeter>`. Ajoutez des propriétés à l'interface `GreeterProps`, puis ajoutez les liaisons nécessaires.
