Les hooks sont un moyen de séparer la logique commune des composants qui l'utilisent. Ils sont un moyen de partager la logique entre les composants. Alors que Qwik fournit beaucoup de hooks, il y en aura toujours un qui ne sera pas fourni dans la boite. Ce tutoriel va vous montrer comment créer votre propre hook.

Dans cet exemple, l'enregistrement des événements `mousemove` est quelque chose qui pourrait être partagé entre plusieurs composants. Refaites le code en retirant le code avant le JSX dans sa propre fonction `useMousePosition()`.

Félicitations, vous avez réussi à créer votre propre hook ! Vous pouvez maintenant l'utiliser dans n'importe quel composant qui a besoin d'écouter la position de la souris.
