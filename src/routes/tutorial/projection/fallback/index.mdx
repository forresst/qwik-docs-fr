Le contenu de secours permet au composant enfant d'afficher un contenu de secours au cas où le composant parent ne fournirait pas de contenu. Le contenu de secours peut être fait avec du CSS, il est nécessaire d'ajouter un sélecteur CSS (et le CSS approprié) sur le Slot nommé au cas où il serait vide.
