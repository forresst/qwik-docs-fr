Utilisez `useOn()` / `useOnDocument()` / `useOnWindow()` pour configurer de manière programmatique des écouteurs pour les éléments hôtes. Ceci est utile lorsque vous créez des API personnalisées et que vous n'avez pas accès à l'emplacement de ces événements dans le JSX ou si les événements ne sont pas connus à l'avance, par exemple s'ils sont créés en fonction des props des composants.
