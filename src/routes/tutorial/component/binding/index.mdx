Le but des composants est de fusionner des données avec le modèle JSX. Vous utilisez `{expression}` pour injecter des données dans un template. Les expressions sont placées soit comme un nœud de texte, soit comme un attribut sur un élément.

- Liez `data.name` à l'attribut `value` de `<input/>`.
- Liez `data.description` au contenu de `<textarea/>`.
