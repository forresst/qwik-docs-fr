Les composants sont combinés entre eux pour créer des applications.

Dans cet exemple, les composants `<App />` et `<Greeter />` sont préparés pour vous. Pour cet exercice, placez le composant `<Greeter />` à l'intérieur du composant `<App />` pour voir son contenu.
