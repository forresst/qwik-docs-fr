Une closure peut être convertie en une référence chargée de manière différée en utilisant la fonction `$()`. Cela génère un type `QRL<Function>`. Une QRL est une référence chargeable de manière différée de la closure. Dans notre cas, nous avons extrait la closure associée à l'événement `onInput` dans le corps du composant. Comme elle n'est plus en ligne, nous devons changer la façon dont le JSX s'y réfère, de `onInput$` vers `onInputQrl`.

Remarquez que notre closure se ferme sur le `store` qui est capturé par l'Optimiseur et ensuite restauré si nécessaire.

### Exemple

Dans cet exemple, nous voulons montrer comment il est facile de charger de manière différée un comportement dans Qwik. Chargeons de manière différée le code qui est exécuté lorsque la touche `Enter` est pressée. Enveloppez le code associé à `Enter` avec `$()` pour le marquer pour le chargement différé. Le `QRL<()=>void>` résultant peut alors être appelé de manière différée avec `.invoke()` et le résultat peut être attendu avec le mot-clé `await`.

Regardez l'onglet `Symbols` pour voir comment le code a été découpé en morceaux.
