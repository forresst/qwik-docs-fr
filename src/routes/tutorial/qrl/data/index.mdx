L'Optimiseur peut être utilisé pour le chargement différé de données, et pas seulement de fonctions. L'avantage de cette méthode est que le chargement différé de n'importe quelle partie de votre application devient facile.

Utilisez la fonction autonome `$()` pour marquer les données que vous souhaitez charger de manière différée. La fonction `$()` renvoie un `QRL` qui est une référence qui peut être sérialisée par Qwik et qui peut plus tard être résolue par la valeur originale.

### Exemple

Chargez les données représentant le message de manière différée au sein de l'écouteur de clic.

Pour cet exercice, imaginez que nous souhaitons charger de manière différée la chaîne `'Hello World!'`. Utilisez la fonction `$()` pour marquer la chaîne comme pouvant être chargée de manière différée. Ensuite, utilisez `await` pour résoudre la valeur du chargeable de manière différée.

Allez dans l'onglet `Symbols` et examinez comment le code a été décomposé en plusieurs parties.
