Bienvenue sur le tutoriel de Qwik. Il s'agit d'une série de courts tutoriels qui vous aideront à démarrer avec Qwik. L'introduction est un ensemble de problèmes qui vous apprendront comment créer un composant, stocker des données, attacher des écouteurs, et récupérer des données du serveur. Alors commençons.

## Création d'un composant

Utilisez l'éditeur à votre droite pour créer une interface utilisateur qui ressemble à ceci :

```
Organisation GitHub: __________
- Qwik
- Partytown
```

Le code HTML de ce composant devrait être le suivant :

```HTML
<div>
  <span>
    Organisation GitHub :
    <input value="BuilderIO" />
  </span>
  <div>
    <ul>
      <li>
        <a href="https://github.com/BuilderIO/qwik">Qwik</a>
      </li>
      <li>
        <a href="https://github.com/BuilderIO/partytown">Partytown</a>
      </li>
    </ul>
  </div>
</div>
```

Vous pouvez cliquer sur le bouton « Montrez-moi » en bas à gauche pour voir la solution complétée.
