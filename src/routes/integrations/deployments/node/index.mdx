# Middleware de Node

Le middleware Node de Qwik City vous permet de connecter Qwik City à un serveur Nodejs qui utilise les fonctionnalités communes du middleware. Certains serveurs Node incluent :

- [Express](https://expressjs.com/)
- [Connect](https://www.npmjs.com/package/connect)
- [Polka](https://www.npmjs.com/package/polka)

## Installation

Pour intégrer l'adaptateur `node`, utilisez la commande `add` :

```shell
npm run qwik add express
```

## Construction de la production

Pour construire l'application pour la production, utilisez la commande `build`, cette commande lancera automatiquement `npm run build.server` et `npm run build.client` :

```shell
npm run build
```

[Lisez le guide complet ici](https://github.com/BuilderIO/qwik/blob/main/starters/adapters/netlify-edge/README.md)

## Déploiement pour le dev

Pour déployer l'application pour le développement :

```shell
npm run deploy
```

## Déploiement de la production

Puisque vous avez choisi Node, vous êtes ici dans la vôtre, après avoir exécuté `npm run build` :

- Le dossier `dist` sera créé en incluant tous les fichiers statiques.
- Le dossier `server` sera créé en incluant tous les fichiers du serveur de node.

Afin de déployer le serveur, vous devez exécuter le fichier `server/entry.express.js` dans le serveur de votre choix.
