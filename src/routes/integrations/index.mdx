# Intégrations

Le CLI de Qwik est livré avec des intégrations qui facilitent la connexion à vos outils et services. 

Une fois que vous avez créé une application Qwik City, exécutez la commande suivante pour ajouter de nouvelles intégrations :

```shell
npm run qwik add
```

Vous serez invité à sélectionner l'intégration que vous souhaitez ajouter. Une fois l'intégration sélectionnée, elle sera ajoutée à votre application et vous pourrez commencer à l'utiliser.
