# Tailwind

Tailwind est un framework CSS qui nous fournit des classes utilitaires à usage unique qui sont pour
la plupart opiniâtre, et qui nous aident à concevoir nos pages Web à partir de nos balisages ou de 
nos fichiers .js/.jsx/.ts/.tsx/.mdx. [Site web de Tailwindcss](https://tailwindcss.com/)

## Utilisation

Vous pouvez ajouter Tailwind facilement en utilisant le script de starter Qwik suivant :

```shell
npm run qwik add tailwind
```

La commande précédente met à jour votre appli et définit la configuration correcte dans `vite.config.ts`.

Il ajoute également de nouveaux fichiers à l'intérieur de votre dossier `src`.

- `postcss.config.js`
- `tailwind.config.js`

## Action nécessaire

Après avoir exécuté la commande avec succès, vous devez ajouter les lignes suivantes
dans votre `global.css`.

```css

# Fichier global.css

@tailwind base;
@tailwind components;
@tailwind utilities;

...des choses...

```
