# Intégrations

Qwik est livré avec des intégrations prêt à l'emploi pour faciliter la connexion à vos services préférés, l'utilisation de vos outils de style ou de test préférés et le déploiement de votre application vers des services communs tels que Cloudflare, Netlify ou Express server, et le [Générateur de site statique (SSG)](/qwikcity/guides/static-site-generation/index.mdx).

Une fois que vous avez créé une application Qwik City, exécutez la commande suivante pour ajouter de nouvelles intégrations :

```shell
npm run qwik add
```

Il vous sera alors demandé de sélectionner l'intégration que vous voulez ajouter. Une fois que vous avez sélectionné une intégration, elle sera ajoutée à votre application et vous pourrez commencer à l'utiliser.
