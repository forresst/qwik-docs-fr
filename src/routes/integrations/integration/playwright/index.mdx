# Playwright

Playwright permet d'effectuer des tests fiables de bout en bout pour les applications Web modernes en prenant en charge tous les moteurs de rendu modernes et en fonctionnant sur plusieurs plateformes. [Site Web de Playwright] (https://playwright.dev/).

## Utilisation

Vous pouvez ajouter Playwright facilement en utilisant le script de démarrage Qwik suivant :

```shell
npm run qwik add playwright
```

La commande précédente ajoute playwright à vos dépendances, ajoute une commande `test.e2e` dans votre section de scripts `package.json` et crée une configuration par défaut dans le fichier `playwright.config.ts` au niveau de la racine de l'application.
Il ajoute également un nouveau fichier dans un dossier `tests` nouvellement créé appelé `example.spec.ts` qui contient quelques cas de test simples.

Pour de plus amples informations, veuillez consulter la [documentation Playwright](https://playwright.dev/docs/intro).
