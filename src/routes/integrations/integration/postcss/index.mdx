# PostCSS

PostCSS permet aux développeurs d'écrire du CSS moderne tout en conservant une compatibilité ascendante et en améliorant les performances. Il analyse le code CSS et le fait passer par une série de plugins qui peuvent modifier, optimiser et étendre le CSS avant de produire le résultat final. [Site Web de PostCSS](https://postcss.org/)

## Utilisation

Vous pouvez ajouter PostCSS à votre projet en utilisant le script de starter Qwik suivant :

```shell
npm run qwik add postcss
```

La commande précédente met à jour votre application et définit la configuration correcte dans `vite.config.ts`.

Elle ajoute également un nouveau fichier dans votre dossier `src`.

- `postcss.config.js`
